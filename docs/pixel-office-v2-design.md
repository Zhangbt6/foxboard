## 一、整体愿景

**莲花池畔的科技楼阁** — 国风水墨 × 像素赛博，碧波连天，灯火通明。

白天是碧波万顷的莲花池公园，夜晚星河倒映水面，楼阁灯笼亮起暖黄光芒。四只狐狸在湖心亭、水上长廊、小桥楼阁间工作，头顶天空盘缓缓旋转——从晨曦到星空，时光流转不息。

## 三、背景系统设计

### 3.1 天空盘 (Sky Disc)

**核心机制**：一张 **圆形全景天空图**，通过 CSS/Canvas 旋转实现时间流转。

| 参数 | 设计 |
|------|------|
| 尺寸 | 2560 × 2560 px 圆形（直径=画布对角线×2，保证旋转不露边） |
| 内容分布 | 0°=正午蓝天白云 → 90°=黄昏晚霞 → 180°=深夜星空银河 → 270°=黎明晨曦 |
| 旋转速度 | 360°/24min（1分钟=1小时游戏时间），可调 |
| 图层位置 | depth: 0（最底层），被建筑/水面遮挡 |
| 裁剪方式 | 背景主体图**天空区域透明**（PNG alpha），天空盘在下面旋转穿透显示 |

**Gemini 生图策略**：
```
生成一张 2560×2560 像素风格圆形天空全景图。
从12点方向顺时针：正午蓝天白云 → 3点方向黄昏橙红晚霞 → 6点方向深夜星空银河（银河横贯，星星闪烁） → 9点方向黎明粉紫晨曦。
风格：像素画风，32px颗粒感，颜色过渡自然柔和。
圆形边缘渐变为透明。包含元素：太阳（正午方向），月亮（深夜方向），流星（随机2-3颗），孔明灯光点散布在黄昏-夜晚区域。
```
